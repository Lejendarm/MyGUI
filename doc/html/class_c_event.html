Detailed Description

Classe héritant de Component qui gère les Event.

La classe possède un pointeur sur le SRender et sur l'entity ainsi qu'un tableau de pointeur sur fonction indexé par EventType.

Les fonctions doivent respecter ce prototype void fct(Entity *, sf::Event &) Elles recevront en premier paramètre l'entitée et en deuxième paramètre l'event SFML
