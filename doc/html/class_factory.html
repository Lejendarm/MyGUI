Detailed Description

Classe qui assemble les components pour préparer les objets demandé.

La Factory build les différents types d'objets avec les component dont ils ont besoin.

Member Function Documentation

Entity * Factory::buildButton ( )

Construit un bouton.

Le label du bouton est centré il est peut être setté par l'intermediaire du component CLabel. Les positions et dimensions du bouton sont spécifié avec les variables x, y, w, h du component CGeometry. Les couleurs sont setté dans le component CColor. Les evenements sont setté avec le component CEvent. Les childrens sont ajouté dans le component CChildren.

Returns
Entity *;

Construit un Cercle.
